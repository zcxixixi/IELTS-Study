# IELTS Listening Notes

## 核心方法：并列结构定位法

### 原理

听力题目的答案几乎全部藏在并列结构里。题目给出的 bullet points / 表格结构 = 听力原文的并列结构。

**做题前**：扫题目，看清有几个并列点，预判信号词。
**做题时**：听到信号词立刻提高注意力，后面跟着的是答案。

---

### 关键信号词

| 信号词 | 作用 |
|--------|------|
| `and` / `and also` | 并列引出下一个答案 |
| `so` / `so for a start` | 引出该段落第一个答案 |
| `then` | 进入下一个并列项 |
| `and whether` | 再引出一个并列问题 |
| `like` / `for example` / `e.g.` | 举例信号，答案就在后面 |

---

### 典型例题分析（Cambridge IELTS 真题 · 护理评估）

**题目结构（bullet points）= 原文并列结构**

> Q1: `time for other responsibilities` + `a ___（break）`
> 原文：*...need time for all the other responsibilities...and also that they sometimes need a **break**.*
> **信号词**：`and also` → 锁定第二个并列点

> Q2: `how much ___ (time) the caring involves`
> 原文：*...they'd want to know the **amount of time** you spend looking after...*
> **替换**：amount of time = how much time；looking after = caring

> Q9: `fuel and ___ (insurance)`
> 原文：*You could, and you can claim for the **insurance** too.*
> **信号词**：`and` → 直接跟答案

> Q10: `help to reduce ___ (stress)`
> 原文：*...give you some advice on how to **minimise** it.*
> **替换**：minimise = reduce

### Part 1 高频同义替换（听力专项）

| 题目用词 | 原文表达 |
|----------|----------|
| good / nice | amazing |
| especially | if you like fish（具体偏好→概括副词）|
| good for sharing | lots of small dishes to share / works really well |
| how much time | amount of time |
| caring | looking after |
| especially difficult | particularly difficult |
| preventing | avoid the possibility of |
| loss of memory | forget |
| fuel | petrol |
| reduce | minimise |
| e.g. | like / for example |
| selection / choice | choice / selection |
| atmosphere | place |

---

## Part 1 做题节奏

1. 看题目，标出所有并列结构和 `and` 的位置
2. 预判每个空的词性（名词/动词/形容词）
3. 听到信号词 → 注意力拉满 → 写答案
4. 并列点之间不乱记，只听信号词后的内容
