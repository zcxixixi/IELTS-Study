# IELTS Listening Notes

## 核心方法：并列结构定位法

### 原理

听力题目的答案几乎全部藏在并列结构里。题目给出的 bullet points / 表格结构 = 听力原文的并列结构。

**做题前**：扫题目，看清有几个并列点，预判信号词。
**做题时**：听到信号词立刻提高注意力，后面跟着的是答案。

---

### 关键信号词

| 信号词 | 作用 |
|--------|------|
| `and` / `and also` | 并列引出下一个答案 |
| `so` / `so for a start` | 引出该段落第一个答案 |
| `then` | 进入下一个并列项 |
| `and whether` | 再引出一个并列问题 |
| `like` / `for example` / `e.g.` | 举例信号，答案就在后面 |

---

### 典型例题分析（Cambridge IELTS 真题 · 护理评估）

**题目结构（bullet points）= 原文并列结构**

> Q1: `time for other responsibilities` + `a ___（break）`
> 原文：*...need time for all the other responsibilities...and also that they sometimes need a **break**.*
> **信号词**：`and also` → 锁定第二个并列点

> Q2: `how much ___ (time) the caring involves`
> 原文：*...they'd want to know the **amount of time** you spend looking after...*
> **替换**：amount of time = how much time；looking after = caring

> Q9: `fuel and ___ (insurance)`
> 原文：*You could, and you can claim for the **insurance** too.*
> **信号词**：`and` → 直接跟答案

> Q10: `help to reduce ___ (stress)`
> 原文：*...give you some advice on how to **minimise** it.*
> **替换**：minimise = reduce

### Part 1 高频同义替换（听力专项）

| 题目用词 | 原文表达 |
|----------|----------|
| good / nice | amazing |
| especially | if you like fish（具体偏好→概括副词）|
| good for sharing | lots of small dishes to share / works really well |
| how much time | amount of time |
| caring | looking after |
| especially difficult | particularly difficult |
| preventing | avoid the possibility of |
| loss of memory | forget |
| fuel | petrol |
| reduce | minimise |
| e.g. | like / for example |
| selection / choice | choice / selection |
| atmosphere | place |

## Part 1 做题节奏

1. 看题目，标出所有并列结构和 `and` 的位置
2. 预判每个空的词性（名词/动词/形容词）
3. 听到信号词 → 注意力拉满 → 写答案
4. 并列点之间不乱记，只听信号词后的内容

---

## 审题七步法（填空题）

| 步骤 | 动作 |
|------|------|
| ① | 先看字数限制（ONE WORD / ONE WORD AND/OR A NUMBER）|
| ② | 扫描标题（时间允许时）|
| ③ | 看空格**前2后2**的词，判断词性 |
| ④ | 介词后→名词；冠词后→名词；动词后→doing/名词 |
| ⑤ | 竖向比较各题，找独特定位词，忽略重复词 |
| ⑥ | 圈专有名词/数字（这些原词重现概率最高）|
| ⑦ | 形容词/动词不稳定，不能依赖原词定位 |

## 七坑表（错题分析框架）

做完题对答案后，对照这七项找出错误原因：

| 坑 | 问题 | 解决方法 |
|----|------|----------|
| ① 词音 | 没听出这个词 | 查音标，朗读10遍 |
| ② 拼写 | 听出来了但写错 | 抄写10遍，边写边读 |
| ③ 句子理解 | 没翻译出答案句 | 把答案句完整翻译+朗读10遍 |
| ④ 逻辑 | 没反应到信号词 | 整理逻辑词，练条件反射 |
| ⑤ 改写 | 没识别出同义替换 | 对照原文找替换对，加入wordbook |
| ⑥ 格式 | 单复数/大小写/连字符错误 | 专项记忆（如10-minute walk）|
| ⑦ 单复数 | 词尾/s/没听清 | 根据题干语法预判，听辨词尾 |

## 逻辑信号词完整表

| 类型 | 信号词 |
|------|--------|
| 并列 | and / and also / then / another thing / also |
| 转折/重点 | but / however / well |
| 因果 | so / because |
| 强调 | particularly / in fact / actually / one thing |
| 举例 | like / for example / such as |
| 纠正 | no / sorry / I mean |
| 序列 | first / firstly / second / finally |

⚠️ **纠正表达后的内容才是正确答案**（no sorry 之后的信息优先级最高）

---

## 陷阱类型

| 陷阱 | 说明 |
|------|------|
| 先提旧信息再转折 | 听到 another / but 之后才是答案 |
| 否定词 | not / no / neither 容易漏听 |
| 比较级 | 给出多个数值，只要最大/最小的那个 |
| 计划 vs 实际 | 区分"打算"和"实际发生"的时态 |
| 数字相似 | fourteen(14) vs forty(40)，听元音长度 |

## 数字专项

| 规则 | 示例 |
|------|------|
| 大数字逗号分段 | 4,483,257 = four million... |
| 1个逗号=thousand，2个=million，3个=billion | |
| double / triple | 221 → two two one 或 double two one |
| 零可读作 oh | 007 → double oh seven |
| 年份简化 | 1900 → nineteen hundred |
| 1500口语 | fifteen hundred |
| 分数 | 2/3 → two thirds（分子>1分母加s）|
| 小数 | 3.423 → three point four two three（逐位读）|

## 格式规范

| 类型 | 正确格式 |
|------|----------|
| 日期 | 17 October 或 October 17（均可）|
| 时间 | 12:30 或 12.30（均可）|
| 复合形容词 | 10-minute walk（minute不加s，需连字符）|
| 大小写 | 不影响得分 |
| 英美拼写 | 均可（enrol/enroll）|
| 括号内容 | 可选填 |
