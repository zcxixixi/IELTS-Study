# IELTS Study

Open-source notes for IELTS preparation.

This repository is built as a long-term study log. It will gradually collect:

- Reading question analysis
- Problem-solving methods
- Common traps and error patterns
- Vocabulary and paraphrase notes

## Structure

- `Reading/`: reading practice notes, question analysis, and strategy documents

## Working Method

When a new question is discussed:

1. Record the passage topic and question type.
2. Explain the solving path step by step.
3. Summarize why the right answer is right.
4. Summarize why the wrong options are wrong.
5. Extract reusable methods into strategy notes.

The goal is not only to get the answer, but to build a reusable reading system.
