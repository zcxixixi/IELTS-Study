# Reading

This folder stores IELTS Reading materials and reusable solving methods.

## Planned Content

- `question-analysis-template.md`: template for future question explanations
- `strategy-notes.md`: reusable methods and recurring traps

## Documentation Rule

Each future analysis should try to answer:

1. What is the question type?
2. What signal words should be located first?
3. Which sentence in the passage is the key evidence?
4. How is the correct answer matched or paraphrased?
5. What trap caused the mistake?
6. What method can be reused next time?
