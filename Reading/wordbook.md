# IELTS Reading Wordbook

This is the main vocabulary book for active memorisation.

Rule:

- add only useful words or phrases worth reusing
- prefer high-frequency IELTS words, research words, logic words, and paraphrase pairs
- do not dump every unfamiliar word into the book
- keep each batch small enough for real review

## Daily Target

Recommended: `8-12` new items per day.

Why:

- fewer than `8` is usually too slow for visible growth
- more than `12-15` often becomes recognition without retention
- `10` is the best default target for steady long-term memory

## Standing Study Rule

This wordbook is the default vocabulary system for IELTS Reading in this workspace.

During study sessions:

- do not wait for the user to open the wordbook first
- proactively review a few older items during normal learning
- then add a small batch of new useful words if the passage contains them
- keep review integrated into the lesson instead of turning it into a separate heavy task

Default session rhythm:

1. review `3-5` old items briefly
2. study the new passage or questions
3. add `5-10` useful new items only if they are worth remembering

## Review Method

For each item, try to recall:

1. the English meaning
2. the Chinese trigger meaning
3. one likely IELTS paraphrase
4. one sentence or context where it appears

## Important Reminder

In IELTS Reading, many research passages use words such as:

- `perceived`
- `rated`
- `judged`
- `seen as`

These usually describe `impression`, not objective fact.
