# 雅思阅读真经派 课程笔记

---

## 一、阅读的两个核心问题

| 问题 | 表现 | 解决方向 |
|------|------|----------|
| 读不懂 | 正确率不高 | 词汇 + 同义替换敏感度 |
| 读不快 | 时间不够 | 变速阅读 + 方法熟练度 |

---

## 二、四大题型分类

| 题型 | 包含 | 核心考点 |
|------|------|----------|
| 单词题 | Summary填空、简答题、Table填表 | 所有需要填词的题型，解题逻辑相同 |
| 句子理解题 | TRUE/FALSE/NOT GIVEN、单选题 | 正确选项 = TRUE的完整表达 |
| 段落题 | Heading题 | 段落中心思想 |
| 全文中心题 | Title题 | 优先看末段（conclusion），或回顾已做题 |

> ⚠️ 全文中心题：**不要**重点读每段首句，这是无效方法。长文章首段通常只是引入，不重要。

---

## 三、词汇分类体系

### 四项对词汇的不同要求

| 技能 | 核心要求 | 备注 |
|------|----------|------|
| 听力 | 常见词的听觉反应速度 | 熟悉才能跟上语速 |
| 阅读 | 认知词汇量的广度 | 模糊认知也能答题 |
| 写作 | 词汇搭配（collocation） | 用错场景直接扣分 |
| 口语 | 自然流畅的表达组合 | 同上 |

> 备考建议：不要用同一本词汇书备考四项，需针对性学习。

### 三大类词汇

| 类别 | 定义 | 处理方式 |
|------|------|----------|
| 模板词 | 题干高频但无定位价值的词，如 account / description / statement / reference | 认识即可，直接跳过不做定位 |
| 主题词 | 文章特定话题的新创词 | 通过上下文推测，无需刻意记忆 |
| 考点词 | 直接影响题目理解的关键词，有特定替换关系 | **最高优先级**，重点背诵 |

### 重点考点词：resemble

- 含义：相似，看起来像
- 同义替换：`resemble = look like = be similar to`
- 出现位置：剑7、剑8、剑9、剑10 均有考察
- 解题逻辑：题干出现 resemble，原文对应 look like / like，后面跟的名词即为答案

### 词汇量标准

- 每篇13道题中出现 **3个以上生词** → 可能错3题 → 加上其他失误 → 单篇错5题 → 三篇错15题 → 对应6~6.5分
- 目标7分：控制题干生词率低于 3/13

## 四、三大类 Keywords（定位词）

### 定位慢的根本原因
1. Keywords 选择不当
2. 定位方法整体错误（读题不够仔细就急于画词）

### 正确读题方法
- 读**两遍**：第一遍理解大意，第二遍抓比较关系/逻辑关系
- **必须把关键信息写下来**，不要只靠记忆

### 三类 Keywords

| 类别 | 特征 | 例子 |
|------|------|------|
| 眼球级 | 最显眼，视觉上最易识别 | 数字、大写专有名词 |
| 朴素级 | 最简单、印象深、同义替换概率小的基础词 | dog / man / cold / foot |
| 逻辑关系词 | 反映句子内在逻辑 | 比较级（higher/lower）、因果词 |

> 真经派核心贡献：**朴素级 keywords** 和**逻辑关系词**是真经派提出的创新定位方法。

### 例题解析（剑7 TRUE/FALSE/NOT GIVEN 第21题）
- 题干："工业世界人均用水量高于古罗马"
- 第一遍：现代与古代用水量的比较
- 第二遍：抓住比较关系词 `higher`
- 定位：寻找古今用水量的直接比较，而非只靠大写字母定位
- 判断：原文有明确比较 → TRUE/FALSE；无比较 → NOT GIVEN

## 五、同义替换总纲标注法

### 核心理论
**题干中每一个词都必然来自原文**，理解题干与原文的对应关系是解题关键。

### 标注流程
1. 划出题干所有关键词
2. 在原文定位每个词的对应表达
3. 确认所有对应关系后再填写答案
4. **必须找到至少两组同义替换才能确定答案**

### 例题（剑8 documentation题）

| 题干词 | 原文对应 |
|--------|----------|
| documentation | written accounts |
| limited | a few incomplete |
| main sources | largely from |

> Heading题：找到两点对应 = 90% 正确，三点对应 = 100% 正确。

### 常见错误
- 仅凭感觉做题
- 不标注直接作答（等于蒙题）
- 追求速度忽视质量

---

## 六、并列结构考点（and）

### 核心原则
题干中出现并列结构，原文中**必然**存在对应并列结构，否则属于篡改原文含义。

### 常见并列词
`and` / `both...and...` / `as well as` / `not only...but also...` / `besides` / `moreover`

### 同义替换链示例
`shifting = change = modify = revision`

### 解题步骤（匹配题）
1. 先不看选项，直接用朴素级词汇 + 并列结构定位原文
2. 找到原文对应处后确定"心目中的答案"
3. 再在选项中找到该答案的同义表达

> 技巧：匹配题随着做题进度，选项范围会自然缩小。

---

## 七、Heading 题做法

### 真经派方法（与主流方法相反）
❌ 不要先做 Heading 题
✅ 正确顺序：**先做自然段内的细节题，完成后再选 Heading**

### 具体步骤
1. 忽略 Heading 选项，直接读文章
2. 先解决该段内的细节题（TRUE/FALSE/填空等）
3. 完成细节题后通读该段，顺势选择 Heading

### 原因
- 细节题考察的句子通常就是段落重要信息
- 通过细节题答案词汇可以直接对应 Heading 选项
- 文章只需读一遍即可完成所有题目

> ⚠️ 反对"首句二句末句"判断中心思想的机械方法——中心句可能在段落中间。

---

## 八、阅读速度方法

### 变速阅读原则

| 内容 | 处理方式 |
|------|----------|
| 读得懂的句子 | 慢读，争取记住 |
| 读不懂的句子 | 加速跳过 |
| 不出题的段落 | 极速略读 |

### 速度目标
- 哈佛研究标准：优秀阅读者达到 **每分钟1000词**
- 考试目标：三篇文章30分钟完成，剩余30分钟检查

### 提速技巧
- 扩大视距，一眼摄入3-4个单词
- 避免默读（默读会将速度降至1/4）
- 像读杂志一样放松地读文章
- 重点抓**名词**和**动词**，其余快速扫过

---

## 九、做题顺序总原则

1. **先读题，后读文章**
2. 标记乱序题（Heading、Matching）关键词
3. 按顺序做其他题型（选择题、填空题）
4. 做题过程中同步解决乱序题

| 题型 | 顺序特点 |
|------|----------|
| Heading题、Matching题 | 乱序为主 |
| 选择题、填空题 | 顺序为主（偶有颠倒） |

---

## 十、学习路径建议

| 阶段 | 做法 |
|------|------|
| 初期 | 完全不卡时间，先确保每个解题动作标准规范 |
| 中期 | 方法熟练后速度自然提升 |
| 禁止 | 方法不成熟时盲目追求速度 |

> 核心逻辑：正确方法 → 提高熟练度 → 自然缩短时间。先追速度会固化错误方法，难以突破瓶颈。

# 第二节课笔记

---

## 一、单词题解题方法

### 题型范围
- Summary填空 / SC句子填空 / SA简答题 / Table表格填空 / Flowchart流程图填空

### 夹心饼干原则
- **考点是空格前后的词**，而非空格本身
- 定位空格两侧的名词/动词在原文中的同义替换
- 遇到生词：用排除法，划掉认识的选项词，剩下的可能就是答案

### 解题技巧
- 每读一段只需记住1-2个关键词，不要试图记忆全部题目
- 生词直接跳过，不纠结
- 注意 `not the only` 这类并列结构提示
- 关注转折词 `but` 后的重要信息

---

## 二、TRUE / FALSE / NOT GIVEN 详解

### TRUE 题命题逻辑

考官出 TRUE 题时**必须**改写多个单词，通常有 **3组及以上同义替换**。

> 判断标准：一句话中出现3组以上同义替换 → 即使有生词也选 TRUE

**经典例题（剑7）：**

| 题干词 | 原文对应 |
|--------|----------|
| Modern technologies | new |
| led to | thanks to |
| reduction | conserve |
| domestic | homes |

→ 四组同义替换，选 TRUE

---

### FALSE 题命题逻辑

FALSE 题只有两种核心设计：

| 类型 | 原理 | 例子 |
|------|------|------|
| 反义词设计 | 原文某词被改成反义词 | 原文 `as much as`，题干改为 `higher` → FALSE |
| 关系性驳斥 | 题干的因果/比较关系与原文不一致 | 原文：事故导致 FAA 成立；题干：喷气引擎导致 FAA 成立 → FALSE |

> ⚠️ 区分 FALSE 与 NOT GIVEN：原文提到了关系但关系不一致 → FALSE；原文根本没有提到关系 → NOT GIVEN

---

### NOT GIVEN 题两种类型

| 类型 | 定义 | 例子 |
|------|------|------|
| 具体型 | 题干中的某个名词/主语/宾语在原文中找不到 | 原文没有提到"调查了蚂蚁的交流" |
| 关系型 | A 找得到，B 也找得到，但两者之间的关系原文没有建立 | 原文分别提到古代和现代水利体系，但没有做比较 |

> 出题规律：一般先出**关系型** NOT GIVEN，再出**具体型** NOT GIVEN

---

### 多选题技巧

多选题答案**必然对应原文中的并列结构**。

**解题步骤：**
1. 定位题干关键词
2. 在原文寻找并列结构（and / or 连接）
3. 将选项与并列成分做同义替换匹配

**剑9 例题：**

| 原文并列结构 | 对应选项 |
|-------------|----------|
| classroom desk organization | — |
| poor acoustics | — |
| mechanical systems (e.g. air conditioning) | C. cooling systems |
| modern educational practices | A. current teaching methods |

---

## 三、同义替换高频词（第二节课补充）

| 含义 | 替换组 |
|------|--------|
| 很久以前 | long before = tens of millions years earlier |
| 导致/因为 | resulting = working on（隐含因果） |
| 类似雷达的 | radar-like = "radar"（带引号） |
| 战争相关 | wartime = military = weapon = World War II |
| 面部视觉 | facial vision = obstacle sensing |

---

## 四、考官出题规律总结

| 题型 | 考官做法 |
|------|----------|
| TRUE | 主动改写多个关键词（≥3组同义替换） |
| FALSE | 找一个重要词改成反义词，或倒置因果/比较关系 |
| NOT GIVEN | 直接添加原文没有的关系，原词可能不改 |

# 第三节课笔记

---

## 一、FALSE 题补充

### 两种核心类型（再次确认）

| 类型 | 原理 |
|------|------|
| 反义词驳斥 | 原文"关心" → 题干改为"漠不关心" |
| 关系型驳斥 | 题干的因果/比较关系与原文不一致 |

### NOT GIVEN 判断补充

- A 和 B 出现在**不同段落**且无明确关系 → NOT GIVEN
- 题干主语或宾语名词在原文**完全未提及** → NOT GIVEN

---

## 二、归纳总结类题目

**特征：** 题干用抽象词汇（如"重要性增长"）概括原文多个具体描述，考查对相邻句群隐含主旨的把握。

**解题方法：**
- 先浏览选项再读段落，带着问题意识寻找线索
- 注意让步结构：`although/while...but...` → 重点在 but 之后，让步部分不是主旨
- 正确答案常包含与原文关键概念的同义替换

---

## 三、单选题解题方法

### 核心原则
- 正确选项 = TRUE 的表达
- 干扰选项分两类：含原文未提及信息（NG）/ 与原文矛盾（FALSE）

### 解题步骤
1. 定位题干关键词（注意：引言部分也可能出题，不要跳过）
2. 找到原文对应句
3. 排除绝对错误选项，锁定答案

### 例题（嗅觉识别实验）

| 选项 | 错因 |
|------|------|
| A. new smell | 原文未提及（NG） |
| B. powerful smell | 与出题点无关（NG） |
| D. surrounded by odors | 与 without 结构矛盾（FALSE） |
| **C. our ability to smell is damaged** | ✅ 对应原文 `only when faculty is impaired` |

**同义替换：** `faculty = ability` / `impaired = damaged`

---

## 四、西方写作段落结构规律

**规律：** 西方写作常在段落首句声明要讨论多个要素，但实际每个要素单独成段，首段只预告不展开。

**解题陷阱：** Heading 题不要被首句误导选包含多个要素的选项，以段落**实际讨论内容**为准。

> 例：剑7 F段首句提到"牙齿和语言两种方法"，但实际只讨论牙齿证据 → Heading 选 "dental evidence" 而非 "dental and linguistic"

---

## 五、匹配题（人名与理论）

### 四种匹配题型

| 类型 | 难度 |
|------|------|
| 关系型匹配（人名-理论、数字-害虫、建筑-特点） | 最常见 |
| 句子型匹配（上半句配下半句） | — |
| 句子补全匹配 | — |
| 图表匹配 | 最少考 |

### 解题方法
1. 题干读**两遍**，对 keywords 留下深刻印象
2. 对人名只需标记位置，**不需要记忆拼写**
3. 定位时**人名 + keywords 结合**，不能只依赖人名
4. 重点寻找 keywords 的同义替换

### 高频同义替换（匹配题）

| 题干词 | 原文对应 |
|--------|----------|
| calculated | determined |
| a fair degree of accuracy | reasonably accurate |
| understood | realized / had shown |
| comparing | different / difference |
| work out | measure = calculate = determine |
| time taken to go around | orbit speeds |
| depends on | governed by |
| witness | saw |
| unable | ruled out |

> ⚠️ 特别注意否定表达：`unable / ruled out / failed to` 等

---

## 六、长难句解析方法

### 核心原则
- 关注**语法结构**，不必纠结每个词的含义
- 找准主谓宾，定语从句修饰的是什么

### 常见结构

| 结构 | 要点 |
|------|------|
| `it was found that...` | it 是形式主语，that 从句是真正主语 |
| `the longer..., the more...` | 前半是从句，后半是主句，more 后接完整句子 |
| `by which... can be judged` | 介词前置的定语从句，还原为 `which...judged by` |
| `which` 引导从句 | 通常是定语从句；状语从句用 when/where/if/although |

### 写作可借鉴表达
- `an increasing awareness and understanding of...`（越来越重视……）
- `there be` 结构：be 动词单复数由后面名词决定

# 第四节课笔记

---

## 一、TRUE / FALSE / NOT GIVEN 深化

### is 与 may 的区别

> 原文：`Tom is a good guy` → 题干：`Tom may be a good guy` → **FALSE**

原文用确定表述，题干改为模糊表述 = FALSE。

**反例（常见错误思路）：**

| 原文 | 题干 | 正确答案 |
|------|------|----------|
| impossible | possible | FALSE |
| cannot | could | FALSE |
| shouldn't | may | FALSE |

> ⚠️ 不能仅凭 `possible / probable / likely / may / could` 等词就选 TRUE，必须找同义替换对应。

---

### NOT GIVEN 的两种细分

| 类型 | 定义 | 例子 |
|------|------|------|
| 关系缺失型 | A 和 B 都有，但关系未建立 | 原文未提 MF 建议 P 参加皇家学院 |
| 宾语缺失型 | 题干的宾语名词在原文根本没出现 | 原文只引用 Pasteur 名言，未提他的 discoveries |

---

### Perkin 文章真题解析

| 题目 | 答案 | 关键逻辑 |
|------|------|----------|
| MF 是否是第一个出现的 | FALSE | 原文先提到了 th |
| MF 是否建议 P 参加皇家学院 | NOT GIVEN | 原文未提及此关系 |
| P 是否雇佣 awh 作为助手 | FALSE | 原文是 P 成为 h 的助手，关系反了 |
| P 年轻时的发现带来名利 | TRUE | `still young` = `not long after that`，`discovery` = `breakthrough` |
| 金鸡纳树只生长在南美 | NOT GIVEN | 原文说 `native to`（原产自），≠ `only grow in` |
| P 用苯胺制造奎宁 | TRUE | 原文直接对应，`hoping to` = `attempting to` |
| Pasteur 的发现启发了 P | NOT GIVEN | 原文只引用 Pasteur 名言，无 discoveries 相关内容 |

**重要名言：** `chance favours only the prepared mind`（机会只青睐有准备的人）—— 剑桥真题出现过两次。

## 二、Heading 题深化

### 三大干扰陷阱

| 陷阱类型 | 说明 |
|----------|------|
| 首句陷阱 | 首句提到 `assumption`，但段落主旨不是 assumption |
| 末句陷阱 | 末句提到 `vast distance`，但不是段落中心 |
| 表面信息陷阱 | 段落表面讲知识局限，实际核心是"可能性" |

### 正确定位段落中心的方法

1. 先做自然段细节题，再选 Heading（细节题已包含段落重要信息）
2. 关注 `however / in fact / but` 等逻辑连接词，这些词后面才是重点
3. 下一段首句常常能揭示上一段的中心思想
4. 已做完的细节题答案词汇可以直接对应 Heading 选项

### 同义替换实例（Heading 题）

`aviation disaster prompts action` 对应原文：
- `aviation` = `skies`
- `disaster` = `accident`
- `prompts` = `resulting`
- `action` = `establishment`

## 三、段落逻辑结构（C 段案例）

**四层递进结构：**

| 层次 | 内容 |
|------|------|
| 第1-3行 | 承认知识有限（铺垫） |
| 第4-5行 | however → 宇宙无限可能性存在（转折，重心） |
| 倒数第3行 | 具体可能性数值（1/100,000） |
| 末句 | 距离推论 |

> 核心考点：段落中出现多个相同数字时，考官考出现次数**少**的那个（更具区分度）。

## 四、单选题补充

### 干扰项识别

| 干扰类型 | 特征 |
|----------|------|
| NG 型干扰 | 包含原文未提及的信息 |
| FALSE 型干扰 | 与原文信息矛盾 |

### 解题步骤
1. 定位题干关键词（引言部分也要读，不要跳过）
2. 找到原文对应句，确认同义替换
3. 排除绝对错误的选项
